# bloco-notas

# Sobre o Project.

Este é um dos meus primeiros projectos com react js, como o nome diz é um bloco de notas, vai ler as notas dos usario(vai receber titulo e descriçao da nota). Após isso vai adicionar a interface.
obs: todas as notas no ui desaparecem quando a pagina é recarregada

## Tecnologias usadas

As tecnologias usadas foram

- HTML / CSS / JS
- React JS
- Vite.

## Como executar o projecto

    ```
    bash
    #Clonar o repositorio
    git clone https://github.com/Marcio-py/bloco-notas.git

    # entrar na pasta do projecto.
    cd bloco-notas

    # Instalar as dependencias necessarias.
    npm i ou npm install

    # Executar o projecto
    npm run dev
    ```
